  • WEB_TITLE
    商城名称：将显示在导航条和标题中
  • WEB_COMPANY
    公司名称：将显示在页面底部公司信息处
  • DOMAIN_NAME
    商城url：将显示在导航条和标题中
  • WEB_ADMINFILE
    设置商城后台访问文件名，默认为“admin.php”，如果您更改了此设置，需要您手动重命名文件
  • WEB_ADMINFILE
    设置商城后台访问文件名，默认为“admin.php”，如果您更改了此设置，需要您手动重命名文件
  • ICP
    页面底部可以显示ICP备案信息，如果网站已备案，在此输入您的备案号，它将显示在页面底部，如果没有请留空
  • COPY
    网站版权信息
  • WEB_ON
    暂时将商城关闭，其他人无法访问，但不影响管理员登录后访问
  • WEB_COUNTCODE
    页面底部可以显示第三方统计，推荐CNZZ,百度统计
  • WEB_LOGO
    填写商城LOGO地址，请用逗号隔开URL，宽度和高度，如：logo.gif,120,60
  • WEB_SUBTITLE
    网页标题通常是搜索引擎关注的重点，本附加字设置出现在标题中商城名称后，如有多个关键字，建议用分隔符分隔
  • WEB_KEYWORDS
    Keywords项出现在页面头部的<Meta>标签中，用于记录本页面的关键字，多个关键字请用分
  • WEB_HEADCODE
    如需在<head></head>中添加其他的HTML代码，可以使用本设置，否则请
  • WEB_POINT_NAME
    可以对消费积分进行重新命名，如：金币
  • WEB_POINT_EXCHANGE
    设置每消费1元可以获得多少积分，默认为没消费1元获得1个积分，如设置为0则关闭消费积分功能
  • WEB_ZONE
    设置系统时区，默认时区：[GMT+08.00] 北京，香港，新加坡，台北
  • WEB_GZIP
    启用Gzip模式可压缩页面大小，加快网页传输，需要php支持Gzip。如果已经用Apache等对页面进行Gzip压缩，请关闭
  • URL_ROUTER_ON
    伪静态是一种搜索引擎优化技术，可以将动态的地址模拟成静态的HTML文件。需要Apache的支持
  • WEB_TITLE
    设置“加入购物车”提示。默认：跳转到购买成功页面库存下降设置
  • WEB_STOCK
    设置库存下降时机，默认为当用户下单成功时商品库存下降，需开启库存管理
  • WEB_UNIT
    设置显示的商品价格格式，%s将被替换为相应的价格数字，默认为：￥%s元
  • WEB_NUMBERFIX
    网页标题通常是搜索引擎关注的重点，本附加字设置出现在标题中商城名称后，如有多个关键字，建议用分隔符分隔
  • WEB_POINT_SETUP
    设置每100个积分可以抵用多少元现金，默认为100元抵用1元现金，关闭此功能请设为0
  • WEB_POINT_LIMIT
    设置每消费100元最多可以使用多少元积分，设为0则不限制，此功能必须开启积分消费功能
  • WEB_BLANCE
    设置是否启用余额支付功能,默认开启
  • WEB_INVOICE
    设置是否启用发票功能,默认开启
  • WEB_INVOICE_CONTENT
    客户要求开发票时可以选择的内容。例如：办公用品。每一行代表一个选项
  • WEB_INVOICE_RATE
    设置开发票的税率，单位为%，要开启发票功能才有效
  • WEB_IS_RETURN
    设置订单完成后是否开启订单退货功能,默认关闭
  • WEB_RUTURN_TIME
    设置订单完成后可以退货的时间限制(单位 /天)
  • WEB_HOT_KEYWORDS
    首页显示的搜索关键字，请用逗号分隔多个关键字
  • WEB_DATE
    使用 yyyy(yy) 表示年，mm 表示月，dd 表示天。如 yyyy-mm-dd 表示 2000-01-01
  • WEB_TIME
    设置时间格式，默认为24小时制
  • WEB_LIST_NUM
    设置商品在列表页单个页面展示数量，此设置必须大于等于1，默认为20
  • WEB_HISTORY_NUM
    设置用户浏览商品时显示最近浏览商品的数量，关闭此功能可设为0，默认为5
  • WEB_COMMENT_NUM
    设置显示商品最新评论的数量，关闭此功能可设为0，默认为5
  • WEB_CONSULT_NUM
    设置显示商品最新咨询的问题数量，关闭此功能课设为0，默认为5
  • WEB_ARTICLE_NUM
    设置文章列表页文章显示的数量，默认为20
  • WEB_GOODS_IMG
    设置显示的购买页面商品图片尺寸，请用逗号隔开宽度和高度，如：280,280
  • WEB_GOODS_THUMB
    设置显示的缩略图尺寸，请用逗号隔开宽度和高度，如：100,100
  • WEB_STYLE
    网站默认主题模板
  • WEB_EXPRESS_NAME

  • WEB_EXPRESS_TEL
    寄件人电话：将显示在快递单寄件人电话栏中
  • WEB_EXPRESS_ADDRESS
